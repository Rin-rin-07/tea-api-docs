# Получить информацию о чае

Возвращает данные о конкретном сорте чая по его уникальному идентификатору.

## Запрос
`GET /teas/{id}`

## Параметры
- `id` (обязательный, целое число): идентификатор чая.

## Пример ответа
```json
{
  "id": 5,
  "name": "Дарджилинг",
  "type": "чёрный",
  "brew_temperature_celsius": 95,
  "steep_time_minutes": 3
}
> Ошибки
404 Not Found: Чай с таким ID не найден.
